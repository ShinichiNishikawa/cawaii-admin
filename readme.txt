== Description ==

This plugin enables you to look the WordPress admin panel cawaii.
You can choose where to locate the menu, how the background looks, and change the color of links, boxes and so on.
You can also change Logos of the admin panel and login page, you can choose which metaboxes to remove from dashboard and post-new pages. 

== Installation ==
1. Upload `cawaii-admin` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. Go to setting page and customize your admin panel

= Functions =
Makes your admin panel cawaii through the functions below.
1. Choose where to locate your menu on the header or default sidebar.
2. Choose background image and whether it would scroll or not.
3. Change color of texts, links, metaboxes and tables.
4. Upload your own logo for admin header.
5. Upload your own logo for login page.
6. Choose metaboxes to remove from dashboard.
7. Choose metaboxes to remove from post/page edit pages.

You can choose whether to use these functionality or wordpress default look.
